# AGENTS.md

## Overview
Static GitHub Pages site with standalone HTML/CSS/JS mini-projects. No build system, tests, or linting.

## Projects
Each subdirectory contains an independent mini-project with `index.html`:
- `coin-toss/`, `dice-tossing/`, `dice-on-cavas/`, `random-verb/`, `password-generator/`, `powerball-number-generator/`, `florida-lottery-generator/`, `abc-generator/`, `bible-memory-verses/`

Root utility HTML: `blood-pressure.html`, `communication_board.html`, `index.html`

## Shared Resources
- `js/` - All JavaScript files (one per app)
- `css/` - All stylesheets (one per app)
- `img/` - Images (dice, coins, etc.)
- `pdf/` - PDF worksheets

## Verification
Open HTML files directly in a browser. No server or build required.

## Git Workflow
Push to `master` branch to deploy to GitHub Pages. No CI or pre-commit hooks.

## Jupyter Notebooks
`Shopping_List.ipynb` and `Room_size_calculator.ipynb` - open in Jupyter or VS Code.
